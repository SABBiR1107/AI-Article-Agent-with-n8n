# 🤖 AI Article Agent with n8n

A full-stack, automated AI agent system that integrates a **Streamlit frontend**, a **FastAPI backend**, and an active **n8n automation workflow** to scrape web articles, run AI summarization and insight extraction via Groq LLM, log execution events in Google Sheets, and send structured markdown summaries via email.

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit (Python-based interactive dashboard)
- **Backend:** FastAPI (High-performance web framework for APIs)
- **Workflow Automation:** n8n (Production workflow engine)
- **AI / LLM Orchestration:** Groq (Llama3 / Mixtral models inside n8n)
- **Web Scraping:** Firecrawl Node
- **Storage / Logging:** Google Sheets API
- **Notifications:** SMTP / Gmail Node

## 🚀 Setup Instructions

Follow these step-by-step instructions to get the backend and frontend up and running locally.

### 1. Environment Setup

1. Navigate to the `backend/` directory:
   ```bash
   cd backend
   ```
2. Copy the `.env.example` file to create a `.env` file:
   ```bash
   cp .env.example .env
   ```
3. Open the `.env` file and replace the placeholder values with your active n8n credentials:
   - `N8N_WEBHOOK_URL`: Your production webhook URL generated by n8n.
   - `N8N_WEBHOOK_SECRET`: The security header token (`x-agent-secret`) matching your n8n webhook configuration.

### 2. Backend Server Setup (FastAPI)

1. Ensure you are in the `backend/` directory.
2. Create and activate a Python virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Start the FastAPI development server:
   ```bash
   uvicorn main:app --host 127.0.0.1 --port 8000 --reload
   ```

### 3. Frontend Dashboard Setup (Streamlit)

1. Open a new terminal and navigate to the `frontend/` directory:
   ```bash
   cd frontend
   ```
2. Create and activate a Python virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Start the Streamlit application:
   ```bash
   streamlit run app.py
   ```
5. The frontend will open in your default browser at `http://localhost:8501`.

## 🧪 Testing the End-to-End System

1. **Open the Streamlit dashboard** (`http://localhost:8501`).
2. **Enter your email** and a valid **article URL** (e.g., a Wikipedia page).
3. **Click "Process Article"**.
4. Observe the frontend processing spinner. Behind the scenes, the FastAPI backend routes the request to n8n, securely handling the secret authentication.
5. Once processed, the frontend will display a **Success UI** alongside the generated Session UUID.
6. Verify your **Google Sheets** for the logged row, and check your **Email Inbox** for the markdown-styled AI analysis report!

## 🤖 n8n Workflow Configuration

To successfully run this project, your active n8n workflow must contain the following components:

1. **Webhook:** Listens for incoming POST requests from FastAPI. Must use **Header Auth** checking the `x-agent-secret` header.
2. **Firecrawl Scrape:** Scrapes the raw text content of the provided `article_url`.
3. **Groq Summary Node:** Generates a concise summary from the scraped text.
4. **Groq Insights Node:** Extracts actionable bullet-point insights and keywords.
5. **Google Sheets:** Authenticates and appends a row containing the Session ID, Email, URL, Summary, and Insights.
6. **Gmail / SMTP:** Composes an HTML/Markdown email containing the summary and insights and sends it to the user.
7. **Webhook Respond:** Returns status metadata back to the FastAPI backend to complete the request lifecycle.
